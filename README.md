# Wolf 'n' Hare #

JS version of classic Russian game Ну погоди, also known as a Eggs.

https://www.youtube.com/watch?v=WR0NIdbxMCY

### Running game ###
To run game you need to run node server

```
#!javascript
node app.js

```

### Generate css / js files ###

```
#!gulp
gulp build
gulp watch

```
